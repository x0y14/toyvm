# ToyVM

## 呼び出し規則
intel記法っぽいものを使います．  
`MOV DEST SRC` // DEST = SRC  
`ADD DEST SRC` // DEST += SRC


## レジスタ
| レジスタ名 | 役割                      |
|-----|-------------------------|
| R10 | 戻り値1                    |
| R11 | 戻り値2(R10だけでは足りなかった場合のみ) |


## 関数の引数
呼び出し前にスタックに逆順に入れてください．(要検証)

## 計算結果
スタックに入れます．
## 関数の戻り値
(レジスタのみを用いる場合)最大2つです．  
1つの場合はR10, 2つの場合は加えてR11を使います．
